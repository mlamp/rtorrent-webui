# Changelog

This project uses **CalVer** (`YYYY.0M.MICRO`) and git tags as the source of truth — see
[`RELEASING.md`](RELEASING.md). Per-release notes are published as
[GitHub Releases](https://github.com/mlamp/rtorrent-webui/releases), generated from the
CalVer tags. The image tag, git tag, and `/api/version` always agree.
