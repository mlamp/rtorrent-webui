# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org), v2.1.

In short: be respectful and constructive. Harassment, personal attacks, and discriminatory
language are not tolerated. Maintainers may remove or reject comments, commits, issues, and
contributions that violate this code, and may ban repeat offenders.

## Reporting

Report unacceptable behavior privately to the maintainers via the repository's **Security →
Report a vulnerability** form (it is private) or a direct message to the maintainer on
GitHub. Reports are handled confidentially.

The full text is at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
